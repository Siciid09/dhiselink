### ## `profiles`
This table stores user profile information, linking to the `auth.users` table. It's the central hub for user data, including roles like admin status.

**Required Fields**
- **`id`**: `uuid` (Primary Key, Foreign Key to `auth.users.id`)
- **`created_at`**: `timestamptz` (Defaults to `now()`)

**Optional Fields**
- **`full_name`**: `text`
- **`email`**: `text`
- **`role`**: `text` (e.g., 'individual', 'organization')
- **`is_admin`**: `bool` (Defaults to `false`)
- **`organization_name`**: `text`
- **`account_type`**: `text`
- `avatar_url`, `bio`, `skills`, `resume_url`, `linkedin_url`, `github_url`, `website_url`, `industry`, `logo_url`, `banner_url`, `location`, `name`, `phone`, `website`, `onboarding_complete`, `professional_title`, `employee_count`, `category`, `tags`, `languages`, `certifications`, `portfolio_url`, `faculties`, `accreditation`, `programs`, `departments`, `courses`, `research_projects`, `notable_alumni`, `department`, `public_services`, `announcements`, `community_focus`, `members_count`, `projects`, `social_links`, `work_experience`, `education`, `mission_vision`, `key_services`, `products_offerings`, `key_contacts`, `awards`, `visibility`, `slug`, `cover_image_url`, `tagline`, `services`, `organization_type`, `organization_subtype`, `year_founded`, `degree`, `experience_level`, `field_of_study`, `years_of_experience`, `operating_regions`
